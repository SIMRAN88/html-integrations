Configuration

Configuration

This class represents the configuration class. Usually used to retrieve configuration properties generated in the backend into the frontend.

Constructor

new Configuration()

Members

_properties: Object private static

Static properties object. Stores all configuration properties. Needed to attribute accessors.

properties: Object private static

Static property. The configuration properties object.

Methods

addConfiguration(properties) static

Adds a properties object to Configuration.properties.

Arguments:
  1. properties (Object)  

    properties to append to current properties.

get(key) → String static

Returns the value of a property key.

Arguments:
  1. key (String)  

    Property key

set(key, value) static

Adds a new property to Configuration class.

Arguments:
  1. key (String)  

    Property key.

  2. value (Object)  

    Property value.

update(key, propertyValue) static

Updates a property object value with new values.

Arguments:
  1. key (String)  

    The property key to be updated.

  2. propertyValue (Object)  

    Object containing the new values.
