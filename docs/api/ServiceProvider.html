  1. ServiceProvider

new ServiceProvider()#

Class representing a serviceProvider. A serviceProvider is a class containing an arbitrary number of services with the correspondent path.

Members

_integrationPath: Stringstatic #

The integration path. Contains the path of the configuration service. Used to define the path for all services.

_listeners: Listenersstatic #

ServiceProvider static listeners.

_parameters: ServiceProviderParametersstatic #

Service provider parameters.
