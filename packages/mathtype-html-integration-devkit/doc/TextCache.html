Class: TextCache

TextCache()

This class represent a client-side text cache class. Contains pairs of strings (key/value) which can be retrieved in any moment. Usually used to store AJAX responses for text services like mathml2latex

Constructor

new TextCache()

(c.f Latex class) or mathml2accessible (c.f Accessibility class).
