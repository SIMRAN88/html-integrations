TextCache

This class represent a client-side text cache class. Contains pairs of strings (key/value) which can be retrieved in any moment. Usually used to store AJAX responses for text services like mathml2latex

Constructor

new TextCache()

(c.f Latex class) or mathml2accessible (c.f Accessibility class).

Members

cache: Array.<String>

Cache array property storing the cache entries.

Methods

get(key) → String

Returns the cache value associated to certain cache key.

Arguments:
  1. key (String)  

    Cache key, usually the service string parameter.

populate(key, value)

This method populates a key/value pair into the this.cache property.

Arguments:
  1. key (String)  

    Cache key, usually the service string parameter.

  2. value (String)  

    Cache value, usually the service response.
